E-commerce project built for GWS with a custom frontend and WooCommerce-backed product and order flows.

[preview](#media-overview)

### Stack

- **UI:** Adobe XD and Next.js.
- **APIs:** REST and WooCommerce.
- **Backend:** Express.js and WooCommerce.

### Implementation

- Converted the Adobe XD design into a responsive Next.js storefront.
- Integrated WooCommerce REST APIs for product and commerce data.
- Used Express.js as a backend layer to handle server-side operations and API communication.
- Structured the UI into reusable components for product sections, content blocks, and commerce interactions.

### Result

The project delivered a custom WooCommerce e-commerce frontend with a cleaner user experience and a backend layer prepared for secure integrations.
