E-commerce project built to support a custom storefront experience with product browsing, cart flows, and Shopify-backed commerce operations.

[preview](#media-overview)

### Stack

- **UI:** React 17, React Router, Redux, Axios, Styled Components.
- **APIs:** REST and Shopify Storefront API.
- **Backend:** Node.js and Express.js.

### Implementation

- Built the frontend using React 17 with route-based navigation through React Router.
- Used Redux to keep commerce state predictable across product, cart, and checkout interactions.
- Integrated REST services and Shopify Storefront API through Axios.
- Created an Express.js backend layer to support custom logic and keep sensitive operations away from the browser.

### Result

The project delivered a custom e-commerce experience connected to Shopify while keeping the frontend flexible and the backend responsible for secure server-side operations.
